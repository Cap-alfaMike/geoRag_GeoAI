# geoRag_GeoAI
Infraestrutura de GeoAI para sistemas GeoRAG, integrando PostgreSQL/PostGIS. Oferece schema lógico otimizado com índices espaciais GIST (R-Tree) e B-Tree, unindo geometrias OGC e metadados JSONB. Essencial para Engenharia de IA que exige recuperação contextual geográfica de alta performance em pipelines de Machine Learning.
